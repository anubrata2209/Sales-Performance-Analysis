## Sales Performance & Customer Insights Analysis

## About the Project
This project is about analyzing sales data to understand how a business is performing.  
I used Python, SQL, and Power BI to clean the data, explore patterns, and create a dashboard.

## Tools Used
- Python (Pandas, Seaborn)
- SQL (MySQL)
- Power BI

## What I Did
- Cleaned and prepared the data using Python  
- Performed exploratory data analysis (EDA)  
- Used SQL to find insights like top customers, sales by region, and trends  
- Built a Power BI dashboard to visualize the results  

## Key Insights
- Top-performing regions identified
- Profit margin analysis
- Customer segmentation

## Conclusion
This project helped me understand how to work with real-world data and turn it into meaningful insights for business decisions.
